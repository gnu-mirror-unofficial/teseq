7 Contact and Information

If you have any questions, comments, suggestions, or bug reports, please take advantage of the mailing list at bug-teseq@gnu.org. To subscribe, send an empty email with the Subject “subscribe” to bug-teseq-request@gnu.org; or use the web interface at http://lists.gnu.org/mailman/listinfo/bug-teseq.

The official website for GNU Teseq is at http://www.gnu.org/software/teseq/. For the latest updates and other information, please check that site (or better yet, stay appraised by subscribing to the mailing list.

GNU Teseq uses Savannah to manage bug-tracking, and to host the development source repository in Mercurial: https://savannah.gnu.org/projects/teseq/.

There is also an IRC channel (#teseq), hosted on irc.freenode.net.
